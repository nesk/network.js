Network.js Make accurate network measures in JavaScript

A JavaScript library to measure various aspects of a connection. It can accurately estimate a bandwidth/latency between a client (using a web browser) and a server (running a specific script).

If you have a really fast connection (like a fiber), you may see low bandwidth measures compared to what your connection can really deliver. This is simply due to my server which only supports a bandwidth up to 11 MBps. Yep, that sucks.
